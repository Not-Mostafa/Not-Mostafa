# Hi, I'm Mostafa
> **Offensive Security Enthusiast | Red Teamer | Computer Engineer**

Welcome to my digital workspace! I'm an Offensive Security Engineer with a relentless curiosity for breaking things to understand how they work. My journey in tech is driven by a passion for cybersecurity, hardware engineering, and building creative software solutions.

### The Cybersecurity Journey
My core focus is on **Offensive Security** and **Red Teaming**. After building a strong defensive foundation as an L1 SOC Analyst, I've pivoted to the offensive side of the landscape. 

I specialize in:
* **Penetration Testing:** Hunting vulnerabilities in web and mobile applications.
* **Red Teaming & Exploit Development:** Simulating real-world attacks, crafting custom exploits, and mastering techniques like SQL injection and advanced reconnaissance.
* **CTFs & Continuous Learning:** I'm a regular on platforms like **TryHackMe** and **Hack The Box**, constantly grinding to keep my skills sharp. (I even built my own automated streak-keeper bot using GitHub Actions to stay on track!)

### Beyond the Terminal: Hardware & Code
I don't just live in the software layer. I love getting my hands dirty with hardware and microcontrollers. 
* **Tinkering:** You'll often find me wiring up **STM32, ESP8266, and Arduino** boards, integrating environmental sensors alongside ST7735 TFT screens, and automating practical, real-world systems.
* **Development:** I enjoy building tools that solve actual problems, from automated bots to mobile applications like the campus scheduling app I developed for my university.
* **Reverse Engineering:** I enjoy deconstructing software—frequently dropping down to assembly level to uncover hidden vulnerabilities, understand complex execution flows, and hunt for unusual bugs.
